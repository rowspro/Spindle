Spindle

Everything on one spindle — an offline music library manager for macOS.

INSTALL WITH HOMEBREW

brew install rowspro/spindle/spindle

Signed & notarized — no Gatekeeper hoops. Or grab the latest release directly (Apple Silicon, Intel and Windows builds).

📥 Inbox review gateEverything you download lands in a quality gate: flags, MusicBrainz completeness checks and a strict approve flow — no errors reach your library.
🩺 Library DoctorFinds artist-spelling variants, misplaced files, messy genres and album oddities. Every fix is one undo away.
🌌 GalaxyYour whole collection as an explorable 3D point map, clustered by genre, artist and era.
🎯 WantlistFollow artists, see discography gaps and keep a shopping list that ticks itself off as albums arrive.
🎧 iPod syncRockbox transfers straight from your library, or iTunes mode with an automatic background ALAC mirror.
↩️ Everything undoableMoves, tags, deletes — one Cmd+Z batch at a time, with a full activity register.
